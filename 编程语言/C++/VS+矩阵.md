# 未定义标识符 “M_PI“
## 描述：
使用C++编译，已经使用
```c++
#include <cmath>
```
但是在使用M_PI时依旧报错说未定义
## 原因
在某些编译器中，cmath库中的 M_PI 是一个条件宏，需要额外定义才能使用
## 解决
在代码的第一行加上
```c++
#define _USE_MATH_DEFINES
```
# “{”: 缺少函数标题(是否是老式的形式表?)
## 描述：
代码Linux转win时可能出现“{”: 缺少函数标题(是否是老式的形式表?)
## 原因：
新版visual studio可能存在
## 解决
右下角LF更改为CRLF
